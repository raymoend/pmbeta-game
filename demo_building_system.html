🎮 How to Use Building System:

  1. Login: Use admin/admin123 or ray/[password]
  2. Go to Buildings: Visit /buildings/
  3. Right-click anywhere on the map to see context menu
  4. Select "Place Building" from context menu
  5. Choose building type and flag color
  6. Click "Build Now" to place the building
  7. Click on buildings to see info or collect revenue
  8. Use "Collect All" button to collect from all buildings
